WSL Ubuntu18.04LTS 更换国内源

1.查看 Linux 版本#

$ lsb_release -a

No LSB modules are available.
Distributor ID: Ubuntu
Description:    Ubuntu 18.04.1 LTS
Release:        18.04
Codename:       bionic

2.备份数据源#

$ sudo cp /etc/apt/sources.list /etc/apt/sources-backup.list

3.更换数据源#

$ sudo vim /etc/apt/sources.list

更换源内容
