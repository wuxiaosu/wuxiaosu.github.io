# Android微信数据库密码计算  

- 查找数据文件路径

```bash
ls -l /data/user/0/com.tencent.mm/MicroMsg/*/EnMicroMsg.db
```

- 查看 uin 值

```bash
cat /data/data/com.tencent.mm/shared_prefs/auth_info_key_prefs.xml
```

- 查看设备 imei 值

```bash
getprop | grep imei
```

- 计算 imei + uin md5

```bash
echo -n "8667640302870921054025641"|md5sum
```

- 密码取前七位
